# 🚗 Smart Parking Management System (C++)

## Features
- Park Vehicle
- Unpark Vehicle
- Search Vehicle
- Parking Status
- Undo Last Parking
- Input Validation
- Current Time Recording

## Technologies
- C++
- OOP
- DSA
- STL (Vector, Stack)

## Data Structures Used
- Vector
- Stack

## Concepts Used
- Object-Oriented Programming
- Linear Search
- Input Validation

## How to Run

Compile:
g++ main.cpp -o parking

Run:
./parking
